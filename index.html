Дипломна робота · Flask + MongoDB Atlas

Автоматизована система формування кастомних рецептів крафту для модпаків Minecraft. Підтримка CraftTweaker (legacy) та Vanilla Data Pack (сучасні версії).

Можливості системи

9 робочих станцій
Верстак, піч, доменна піч, коптильня, багаття, кам'яноріз, стіл коваля, ковадло, алхімічна стійка.
Реальні текстури
PNG-іконки Minecraft замість емодзі. Зручний пікер предметів з пошуком та категоріями.
Живий перегляд коду
ZenScript для CraftTweaker або JSON для Data Pack — генерується в реальному часі.
Генерація ZIP
Готовий до встановлення модпак одним кліком — для будь-якої версії гри.
Ролі та права
Адміністратор бачить статистику всіх користувачів; звичайний user — лише свої рецепти.
Всі версії гри
Від 1.7.10 до 1.21.4 — базові предмети доступні незалежно від обраної версії.

🛒 Кошик рецептів

Додайте кілька рецептів і завантажте їх одним ZIP-архівом. Усі рецепти мають бути однієї версії гри.

✅ Архів завантажено! Як встановити:

1.7–1.12 (CraftTweaker)
1.13+ (Data Pack)
  1. Встановіть мод CraftTweaker для вашої версії гри (з CurseForge).
  2. Розпакуйте завантажений ZIP-архів.
  3. Скопіюйте файл scripts/modpack_recipes.zs у теку:
    .minecraft/scripts/
  4. Запустіть гру або введіть команду /reload у грі.
  5. Готово! Ваші рецепти додані до гри.
  1. Відкрийте свій ігровий світ (одиночна гра).
  2. Не розпаковуючи ZIP-архів, скопіюйте його у теку:
    .minecraft/saves/<ваш_світ>/datapacks/
  3. У грі введіть команду /reload.
  4. Перевірте: /datapack list — ваш пакет має бути у списку.
  5. Готово! Рецепти працюють у цьому світі.
